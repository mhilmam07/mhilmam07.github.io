What I'm Good at


GeoSpatial Analysis

Geographic Information System (GIS) adalah suatu sistem informasi yang dirancang untuk bekerja dengan data yang bereferensi spasial atau berkoordinat geografi. GeoSpatial Analysis dapat digunakan untuk menganalisa, memasukan, menyimpan, memperbaiki, memperbaharui, mengelola, memanipulasi, meng-integrasikan, dan menampilkan data dalam suatu informasi berbasis geografis.
Dengan bantuan Software GIS seperti ArcGIS, QGIS dan lainnya saya dapat melakukan analisa ataupun prediksi berdasarkan data yang ada.

Data Analytics

Data Analytics merupakan proses inspeksi serangkaian data yang dilakukan untuk mendapatkan kesimpulan dari informasi yang ada. saya seringkali melakukan analisa terhadap data yang saya peroleh dari website seperti kaggle.com dan lainnya.
saya biasa melakukan analisa dan pengolahan data dengan menggunakan beberapa software database seperti Microsoft Excell, Rstudio dan lainnya. Dengan software tersebut saya dapat melakukan analisa terhadap data yang secara garis besar meliputi eksplorasi data analisis (EDA) yang berguna untuk mencari pola dan hubungan di dalam data dan Confirmatory Data Analysis (CDA) yang berguna dalam tekhnik statistik untuk melihat hipotesis terkait benar atau salahnya informasi dari suatu data.

Query Database

Sebelum melakukan analisa pada suatu Data, perlu dilakukan persiapan dan cleansing data. Untuk melakukan cleansing data dengan baik, saya seringkali menggunakan bahasa pemrograman Query database seperti SQL dengan bantuan software MySQL, XAMPP ataupun SPSS. persiapan dan clensing data dilakukan untuk memastikan bahwa informasi pada data dapat dengan mudah ditarik atas permintaan serta dianalisis dengan benar.

Data Visualization

Visualisasi data dilakukan untuk memudahkan informasi diterima banyak orang. Setelah melakukan analsis dan eksplorasi data, saya membuat visualisasi dari data tersebut dengan menggunakan software Tableau, Meta Base atau Microsoft Power BI. dengan bantuan software tersebut saya dapat lebih mudah untuk membuat dashboard laporan berdasarkan data sehingga dapat dipahami dan digunakan untuk analisa lebih lanjut.
